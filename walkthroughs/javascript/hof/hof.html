Higher-Order Functions

Functions that take in a function value as a parameter, or return a function value are known as Higher-Order Functions.

View the output in the DevTools Console by pressing Fn + F12, or right click and 'inspect the page'.

Taking a function value as a parameter

Syntax:


                        function useContacts(list, action) {
                            for (let value of list) {
                                action(value.age, value.name);
                            }
                        }

                        useContacts(contacts, console.log);

Returning a function value

Syntax:


                        function calcAge(unit) {
                            return function (age) {
                                return age * unit;
                            };
                        }

                        let calcAgeInDays = calcAge(365.25);
