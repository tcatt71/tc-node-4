JavaScript Loops

View the output in the DevTools Console by pressing Fn + F12, or right click and 'inspect the page'.

For Loop

Syntax:


            for (let i = 0; i < 10; i++) {
              console.log(i);
            }

While Loop

Syntax:


            while (isOnline) {
              console.log("User is online. Transmitting data...")
            }

Do While Loop

Syntax:


            do {
              console.log("Sending message...")
            }
            while (!messageReceived);
