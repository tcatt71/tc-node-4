JavaScript Control Flow

View the output in the DevTools Console by pressing Fn + F12, or right click and 'inspect the page'.

Selection Statements

Syntax:


            if (userName == "Ben") {
              alert("Hello Ben!");
            } else if (userName) {
              alert("Hello not Ben!");
            } else {
              alert("You didn't give me a name :(");
            }

Switch Case

Syntax:


            switch (url) {
              case url == ".html":
                console.log("html file");
              case url == ".css":
                console.log("css file");
              case url == ".js":
                console.log("js file");
              case url == ".png":
                console.log("png file");
              case url == ".jpeg":
                console.log("jpeg file");
              case url == ".jpg":
                console.log("jpg file");
                break;
              default:
                console.log("Not a valid file type");
            }
