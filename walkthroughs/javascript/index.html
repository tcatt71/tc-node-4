JavaScript Objects

View the output in the DevTools Console by pressing Fn + F12, or right click and 'inspect the page'.

Arrays

Syntax:


            let scores = [12, 16, 17, 18];

            // or

            let names = new Array("Frodo", "Bilbo", "Merry", "Pippen", "Sam")

Objects

Syntax:


            let person = {
              firstName: "Frodo",
              lastName: "Baggins",
              age: 51
            }

Accessing Values

You can access values of objects and arrays by using Bracket Notation or Dot Notation.

Syntax:


            // Bracket Notation
            scores[2]; // 17
            
            // Dot Notation
            person.firstName; // "Frodo"
