Flexbox

Flexbox is useful for single directional, responsive layouts.

Properties

  • display: flex;
  • flex-direction: [row|column|row-reverse|column-reverse];
  • justify-content: [...];
  • align-items: [...];
  • align-self: [...];
  • flex-wrap: [wrap|nowrap];
  • flex-grow: [1|...];
  • flex-shring: [1|...];
  • flex-basis: [200px|...];

Grid

Grid is useful for multi-dimensional, responsive layouts.

Properties

  • display: grid;
  • grid-template-columns: [1fr|...];
  • grid-template-rows: [1fr|...];
  • grid-column-start: [1|...];
  • grid-column-end: [1|...];
  • grid-row-start: [1|...];
  • grid-row-end: [1|...];
