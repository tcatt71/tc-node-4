# Review

## Switch Cases

```js
switch (expression) {
  case match1:
    // do something
    break;
  //...
  default:
  // do something if no case is matched
}
```

## Loops

- for

```js
for (initialization; conditional; iterator) {
  // code to repeat
}
```

- while

```js
while (condition) {
  // code to repeat
}
```

- do while loop

```js
do {
  // code to repeat
} while (condition);
```
